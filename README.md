Rumus Metode Secant: <br>
xi+1 = xi - [f(xi) * (xi-1 - xi)] / [f(xi-1) - f(xi)] <br><br>

Rumus Et: <br>
Et = (x_asli - xi) / x_asli <br><br>

Rumus Ea: <br>
Ea = (xi-1 - xi) / xi-1
